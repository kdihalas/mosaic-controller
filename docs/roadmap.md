# Roadmap

Deferred work includes managed `Kustomization` objects, Git and Bucket sources, a Mosaic registry or OCI client, external values and imports, progressive delivery, Helm output, remote clusters, admission or conversion webhooks, multi-replica artifact serving, remote workers, distributed storage, package search, automatic environment discovery, and automatic Flux installation.

Cross-namespace sources can be considered only when supported source kinds expose an explicit Flux-compatible object ACL. Multi-replica serving requires shared storage and a separate availability design.
